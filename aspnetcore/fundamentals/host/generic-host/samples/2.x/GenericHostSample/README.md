# <a name="aspnet-core-generic-host-sample"></a>ASP.NET Core 일반 호스트 샘플

이 샘플은 ASP.NET Core 일반 호스트 [HostBuilder](https://docs.microsoft.com/dotnet/api/microsoft.extensions.hosting.ihostedservice)를 보여줍니다. 이 샘플은 [ASP.NET Core 일반 호스트](https://docs.microsoft.com/aspnet/core/fundamentals/host/generic-host) 항목에 설명된 기능을 보여줍니다.
