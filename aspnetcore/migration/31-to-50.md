## <a name="update-net-core-sdk-version-in-globaljson"></a><span data-ttu-id="7b24d-111">global.json에서 .NET Core SDK 버전 업데이트</span><span class="sxs-lookup"><span data-stu-id="7b24d-111">Update .NET Core SDK version in global.json</span></span>

<span data-ttu-id="7b24d-112">[전역 json](/dotnet/core/tools/global-json) 파일을 사용 하 여 특정 .NET Core SDK 버전을 대상으로 하는 경우 설치 된 `version` .net 5.0 SDK 버전으로 속성을 업데이트 합니다.</span><span class="sxs-lookup"><span data-stu-id="7b24d-112">If you rely upon a [global.json](/dotnet/core/tools/global-json) file to target a specific .NET Core SDK version, update the `version` property to the .NET 5.0 SDK version that's installed.</span></span> <span data-ttu-id="7b24d-113">다음은 그 예입니다.</span><span class="sxs-lookup"><span data-stu-id="7b24d-113">For example:</span></span>

```diff

  "sdk": {
-    "version": "3.1.200"
+    "version": "5.0.100-preview.3.20216.6"
  }
}
```

## <a name="update-the-target-framework"></a><span data-ttu-id="7b24d-114">대상 프레임 워크 업데이트</span><span class="sxs-lookup"><span data-stu-id="7b24d-114">Update the target framework</span></span>

<span data-ttu-id="7b24d-115">프로젝트 파일에서 [대상 프레임 워크 모니커 (TFM)](/dotnet/standard/frameworks) 를로 `net5.0`업데이트 합니다.</span><span class="sxs-lookup"><span data-stu-id="7b24d-115">In the project file, update the [Target Framework Moniker (TFM)](/dotnet/standard/frameworks) to `net5.0`:</span></span>

```diff

  <PropertyGroup>
-    <TargetFramework>netcoreapp3.1</TargetFramework>
+    <TargetFramework>net5.0</TargetFramework>
  </PropertyGroup>

</Project>
```

## <a name="update-package-references"></a><span data-ttu-id="7b24d-116">패키지 참조 업데이트</span><span class="sxs-lookup"><span data-stu-id="7b24d-116">Update package references</span></span>

```diff
+    <PackageReference Include="Microsoft.AspNetCore.Mvc.NewtonsoftJson" Version="5.0.0-preview.3.20215.14" />
+    <PackageReference Include="Microsoft.Extensions.Caching.Abstractions" Version="5.0.0-preview.3.20215.2" />

```
