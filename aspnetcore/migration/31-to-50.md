## <a name="update-net-core-sdk-version-in-globaljson"></a><span data-ttu-id="34765-111">global.json의 .NET Core SDK 버전 업데이트</span><span class="sxs-lookup"><span data-stu-id="34765-111">Update .NET Core SDK version in global.json</span></span>

<span data-ttu-id="34765-112">특정 .NET Core SDK 버전을 대상으로 하는 [전역 json](/dotnet/core/tools/global-json) 파일을 사용 하는 경우 `version` 속성을 설치 된 .NET 5.0 SDK 버전으로 업데이트 합니다.</span><span class="sxs-lookup"><span data-stu-id="34765-112">If you rely upon a [global.json](/dotnet/core/tools/global-json) file to target a specific .NET Core SDK version, update the `version` property to the .NET 5.0 SDK version that's installed.</span></span> <span data-ttu-id="34765-113">다음은 그 예입니다.</span><span class="sxs-lookup"><span data-stu-id="34765-113">For example:</span></span>

```diff
{
  "sdk": {
-    "version": "3.1.200"
+    "version": "5.0.100-preview.1.20155.7"
  }
}
```

## <a name="update-the-target-framework"></a><span data-ttu-id="34765-114">대상 프레임 워크 업데이트</span><span class="sxs-lookup"><span data-stu-id="34765-114">Update the target framework</span></span>

<span data-ttu-id="34765-115">프로젝트 파일에서 [대상 프레임 워크 모니커 (TFM)](/dotnet/standard/frameworks) 를 `netcoreapp5.0`로 업데이트 합니다.</span><span class="sxs-lookup"><span data-stu-id="34765-115">In the project file, update the [Target Framework Moniker (TFM)](/dotnet/standard/frameworks) to `netcoreapp5.0`:</span></span>

```diff
<Project Sdk="Microsoft.NET.Sdk.Web">

  <PropertyGroup>
-    <TargetFramework>netcoreapp3.1</TargetFramework>
+    <TargetFramework>netcoreapp5.0</TargetFramework>
  </PropertyGroup>

</Project>
```

## <a name="update-package-references"></a><span data-ttu-id="34765-116">패키지 참조 업데이트</span><span class="sxs-lookup"><span data-stu-id="34765-116">Update package references</span></span>

<span data-ttu-id="34765-117">프로젝트 파일에서 각 `Microsoft.AspNetCore.*`를 업데이트 하 고 패키지 참조의 `Version` 특성 `Microsoft.Extensions.*` 5.0.0 이상으로 업데이트 합니다.</span><span class="sxs-lookup"><span data-stu-id="34765-117">In the project file, update each `Microsoft.AspNetCore.*` and `Microsoft.Extensions.*` package reference's `Version` attribute to 5.0.0 or later.</span></span> <span data-ttu-id="34765-118">다음은 그 예입니다.</span><span class="sxs-lookup"><span data-stu-id="34765-118">For example:</span></span>

```diff
<ItemGroup>
+    <PackageReference Include="Microsoft.AspNetCore.Mvc.NewtonsoftJson" Version="3.1.2" />
+    <PackageReference Include="Microsoft.Extensions.Caching.Abstractions" Version="3.1.2" />
-    <PackageReference Include="Microsoft.AspNetCore.Mvc.NewtonsoftJson" Version="5.0.0-preview.1.20124.5" />
-    <PackageReference Include="Microsoft.Extensions.Caching.Abstractions" Version="5.0.0-preview.1.20120.4" />
</ItemGroup>
```
