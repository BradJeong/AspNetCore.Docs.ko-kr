## <a name="call-configurekestrel-instead-of-usekestrel"></a><span data-ttu-id="1b913-121">UseKestrel 대신 ConfigureKestrel 호출</span><span class="sxs-lookup"><span data-stu-id="1b913-121">Call ConfigureKestrel instead of UseKestrel</span></span>

```csharp
public static IWebHostBuilder CreateWebHostBuilder(string[] args) =>
    WebHost.CreateDefaultBuilder(args)
        .UseStartup<Startup>()
        .ConfigureKestrel((context, options) =>
        {
            // Set properties and call methods on options
        });
```
