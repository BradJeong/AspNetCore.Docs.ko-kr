## <a name="update-target-framework-moniker-tfm"></a><span data-ttu-id="3277a-106">TFM(대상 프레임워크 모니커) 업데이트</span><span class="sxs-lookup"><span data-stu-id="3277a-106">Update Target Framework Moniker (TFM)</span></span>

## <a name="adopt-the-iis-in-process-hosting-model"></a><span data-ttu-id="3277a-110">IIS에서 in-process 호스팅 모델을 채택</span><span class="sxs-lookup"><span data-stu-id="3277a-110">Adopt the IIS in-process hosting model</span></span>

## <a name="update-net-core-sdk-version-in-globaljson"></a><span data-ttu-id="3277a-113">global.json에서 .NET Core SDK 버전 업데이트</span><span class="sxs-lookup"><span data-stu-id="3277a-113">Update .NET Core SDK version in global.json</span></span>
