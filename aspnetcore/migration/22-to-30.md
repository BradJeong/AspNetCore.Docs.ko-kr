<span data-ttu-id="99540-111">솔루션이 [global.json](/dotnet/core/tools/global-json) 파일을 사용하여 특정 .NET Core SDK 버전을 대상으로 `version` 하는 경우 해당 속성을 컴퓨터에 설치된 3.0 버전으로 업데이트합니다.</span><span class="sxs-lookup"><span data-stu-id="99540-111">If your solution relies upon a [global.json](/dotnet/core/tools/global-json) file to target a specific .NET Core SDK version, update its `version` property to the 3.0 version installed on your machine:</span></span>

    "version": "3.0.100"
