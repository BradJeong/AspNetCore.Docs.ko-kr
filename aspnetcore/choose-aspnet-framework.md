# <a name="choose-between-aspnet-and-aspnet-core"></a>ASP.NET 및 ASP.NET 핵심 중에서 선택 

ASP.NET을 솔루션에 만드는 웹 응용 프로그램에 관계 없이: 엔터프라이즈 웹 응용 프로그램에서 Windows Server를 대상으로 작은 마이크로 서비스를 대상으로 Linux 컨테이너 및 모든 것 사이 있습니다.

## <a name="aspnet-core"></a>ASP.NET Core

ASP.NET Core는 Windows, Mac 또는 Linux에서 최신 클라우드 기반 웹 응용 프로그램을 구축 하기 위한 새로운 오픈 소스, 크로스 플랫폼.NET framework입니다.

## <a name="aspnet"></a>ASP.NET

ASP.NET은 엔터프라이즈 수준의.NET을 사용 하 여 windows 서버 기반 웹 응용 프로그램을 구축 하는 데 필요한 모든 서비스를 제공 하는 완성도 높은 웹 플랫폼입니다.

## <a name="which-one-is-right-for-me"></a>어느 내게 적합 한가?

| ASP.NET Core | ASP.NET |
|---|---|
|Windows, Mac 또는 Linux 용 빌드|Windows 용 빌드|
|사용 하 여 [MVC](mvc/overview.md), 또는 [웹 API](tutorials/first-web-api.md)|Use [Web Forms](https://docs.microsoft.com/aspnet/web-forms), [SignalR](https://docs.microsoft.com/aspnet/signalr), [MVC](https://docs.microsoft.com/aspnet/mvc), [Web API](https://docs.microsoft.com/en-us/aspnet/web-api/), or [Web Pages](https://docs.microsoft.com/aspnet/web-pages)|
|컴퓨터 마다 여러 버전|컴퓨터당 한 버전|
|Visual Studio 또는 C를 사용 하 여 Visual Studio 코드를 사용 하 여 개발#|C#, VB 또는 F를 사용 하 여 Visual Studio를 사용 하 여 개발#|
|새 플랫폼|완성도 높은 플랫폼|
|울트라 성능|고성능|
|[.NET Framework 또는.NET Core 런타임을 선택 합니다.](https://docs.microsoft.com/dotnet/articles/standard/choosing-core-framework-server)|.NET Framework 런타임 사용|
