# <a name="aspnet-core-integration-testing-sample"></a>ASP.NET Core 통합 테스트 샘플

이 샘플은 [ASP.NET Core Razor Pages](https://docs.microsoft.com/aspnet/core/mvc/razor-pages) 앱의 통합 테스트 과정을 보여 줍니다. 이 샘플은 [ASP.NET Core의 통합 테스트](https://docs.microsoft.com/aspnet/core/test/integration-tests)에 설명된 기능을 보여 줍니다.

기본 제공 테스트 기능에 IDE를 사용하지 않는 경우 *tests/RazorPagesProject.Tests* 폴더의 명령 프롬프트에서 다음 명령을 실행합니다.

```console
dotnet test
```
