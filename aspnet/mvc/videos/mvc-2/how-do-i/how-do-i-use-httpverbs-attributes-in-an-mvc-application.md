====================
으로 [Chris Pels](https://twitter.com/chrispels)

이 비디오 Chris Pels HttpVerbs 특성을 사용 하 여 MVC 동작에 대 한 액세스를 제어 하는 방법을 보여 줍니다. 먼저, 샘플 응용 프로그램 기본 컨트롤러와 뷰 정보를 편집 하기 위해 생성 됩니다. 다음으로, 두 번째 인덱스 작업에 HTTP POST를 사용할 때만 호출 하도록 제한 HttpPost 특성이 컨트롤러에 추가 됩니다. 추가 작업으로 AcceptVerbs() 특성 Visual Studio 2008에 대 한 대체 구문으로 구현 됩니다. 사용 하 여 HttpVerbs 링크에서 삭제를 수행 하는 HTTP GET 사용과 관련 된 보안 위험을 방지 하기 위한 다음 설명 합니다.
