====================
으로 [Joe Stagner](https://github.com/JoeStagner)

표준 ASP.NET 유효성 검사 컨트롤 서버와 클라이언트 모두에서 작동 합니다. 서버 쪽 유효성 검사는 웹 응용 프로그램 사용자에 게 친숙함에 클라이언트 쪽 유효성 검사를 추가 하는 동안 웹 응용 프로그램의 보안을 추가 합니다. 이 비디오에서 ASP.NET AJAX 컨트롤 Toolkit ValidatorCallout 컨트롤 깔끔하게 설명선 상자에서 유효성 검사 메시지를 배치 하 여 사용자 환경을 향상 시키기 위해 표준 유효성 검사 컨트롤을 확장 하는 방법을 보여 줍니다.
