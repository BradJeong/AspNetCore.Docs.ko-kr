<a name="how-do-i-create-a-custom-profile-provider"></a>[어떻게 할까요?] 사용자 지정 프로필 공급자 만들기
====================
으로 [Chris Pels](https://twitter.com/chrispels)

이 비디오를 저장 하 고 표준 ASP.NET 멤버 자격 공급자의 aspnet에서 별도 SQL Server 테이블에서 사용자 프로필에 연결 된 데이터를 검색할 사용자 지정 프로필 공급자를 만드는 방법을 설명\_프로필 테이블입니다. 이 비디오에서 설명 하는 원칙 액세스 및 Oracle과 같은 다른 데이터베이스에 대 한 프로필 공급자를 만드는 방법에 적용 됩니다. 사용자 지정 프로필 공급자를 만드는 방법을 학습 한 후 웹 사이트의 공급자를 설치 하 고 로드 하 고 프로필 데이터를 저장의 데모를 보려면 하는 방법을 배울 합니다.
