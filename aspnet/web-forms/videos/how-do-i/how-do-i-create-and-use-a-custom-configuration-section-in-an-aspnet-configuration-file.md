<a name="how-do-i-create-and-use-a-custom-configuration-section-in-an-aspnet-configuration-file"></a>[어떻게 할까요?] 만들기 및 ASP.NET 구성 파일에 사용자 지정 구성 섹션을 사용 합니다.
====================
으로 [Chris Pels](https://twitter.com/chrispels)

이 비디오에서는 Chris Pels XML 구성 요소를 사용 하는 ASP.NET 구성 설정의 표준 집합을 확장 하는 방법을 표시 됩니다. 먼저 해석 하 고 사용자 지정 XML 구성 요소에 대 한 정보를 처리 하는 데 필요한 구성 섹션 처리기를 만드는 방법을 참조 하십시오. 이 프로세스 동안 구성 섹션 처리기의 동작을 정의 하기 위한 선언적 모델을 사용 하는 방법을 알아봅니다. 다음으로 ASP.NET 구성 파일을 사용자 지정 섹션 처리기 및 사용자 지정 요소를 추가 하는 방법을 참조 하십시오. 마지막으로, 사용자 지정 구성 데이터를 프로그래밍 방식으로 액세스 하는 방법을 참조 합니다.
