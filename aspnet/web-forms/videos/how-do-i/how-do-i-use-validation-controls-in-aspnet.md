<a name="how-do-i--use-validation-controls-in-aspnet"></a>[어떻게 할까요?]  유효성 검사 컨트롤을 사용 하 여 ASP.NET에서?
====================
으로 [Chris Pels](https://twitter.com/chrispels)

이 비디오에서는 ASP.NET에서 사용할 수 있는 유효성 검사 컨트롤을 사용 하 여 웹 페이지에 유효성 검사를 구현 하는 방법을 알아봅니다. 모든 주요 유효성 검사 RequiredFieldValidator 및 RegularExpressionValidator 대해서는 설명 등을 제어 합니다. 또한 표준 유효성 검사기 중 하나가 요구 사항에 맞지 않을 때 CustomValidator를 사용 하는 방법을 알아봅니다. 서버와 클라이언트 쪽 유효성 검사 옵션을 모두 표시 됩니다. UI 컨트롤에 유효성 검사 뿐 아니라 여러 유효성 검사 단일 수행 하는 방법을 참조 하십시오.
