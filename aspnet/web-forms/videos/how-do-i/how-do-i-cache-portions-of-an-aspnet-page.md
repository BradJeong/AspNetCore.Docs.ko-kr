<a name="how-do-i-cache-portions-of-an-aspnet-page"></a>[어떻게 할까요?] ASP.NET 페이지의 부분 캐시
====================
으로 [Chris Pels](https://twitter.com/chrispels)

이 비디오 Chris Pels OutputCache 페이지에서 사용자 정의 컨트롤을 구현 하 여 ASP.NET 페이지의 여러 부분을 캐시 하는 방법을 보여 줍니다. 첫째, ASP.NET 페이지 만들어지고 OutputCache 지시문을 사용 하 여 캐시 합니다. 그런 다음 몇 가지 사용자 정의 컨트롤 생성 및 서로 다른 기간에 대 한 캐시 됩니다. 페이지 간의 관계 기간을 캐시 하 고 캐시 기간 제어를 검토 합니다. 캐시 컨트롤에 대 한 여러 조건은 VaryByControl 여기서 중첩 된 사용자 정의 컨트롤의 부모 컨트롤의 캐싱을 제어할 수의 예를 포함 하 여 검토 합니다.
