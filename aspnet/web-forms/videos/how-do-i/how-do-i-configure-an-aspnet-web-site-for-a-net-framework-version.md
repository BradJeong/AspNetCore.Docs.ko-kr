<a name="how-do-i-configure-an-aspnet-web-site-for-a-net-framework-version"></a>[어떻게 할까요?] .NET Framework 버전에 대 한 ASP.NET 웹 사이트를 구성 합니다.
====================
으로 [Chris Pels](https://twitter.com/chrispels)

이 비디오 Chris Pels에서는 ASP.NET/.NET 프레임 워크의 특정 버전을 사용 하도록 ASP.NET 웹 사이트를 구성 하는 방법을 보여줍니다. 먼저 웹 사이트가 인터넷 정보 서버 (IIS)에서 스크립트 맵을 확인 하 여 연결 된.NET Framework의 버전을 확인 하는 방법을 참조 하십시오. 스크립트 매핑 IIS에서 루트 및 다음 개별 웹 사이트에 연결 된 방법을 참조 하십시오. ASP.NET IIS 등록 도구에 대 한 자세한 내용은 다음 (aspnet\_regiis.exe) 특정 버전의.NET Framework를 사용 하는 IIS 또는 ASP.NET 웹 사이트를 구성 하는 데 사용 합니다. IIS 기본값 보다 다른 버전의.NET Framework를 사용 하도록 단일 웹 사이트를 구성 하는 방법을이 내용은 참조 하십시오 사용 하 여. 마지막으로,.NET Framework 2.0 및 1.1의 다른 한는 실행 동일한 IIS에서 두 개의 서로 다른 웹 사이트를 참조 하십시오.
