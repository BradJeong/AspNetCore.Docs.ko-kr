<a name="how-do-i-use-the-ftpwebrequest-class-to-create-an-ftp-connection-to-a-ftp-server"></a>[어떻게 할까요?] FtpWebRequest 클래스를 사용 하 여 FTP 서버에 FTP 연결을 만들려면
====================
으로 [Chris Pels](https://twitter.com/chrispels)

이 비디오 Chris Pels FtpWebRequest 클래스를 사용 하 여 FTP 연결을 만들고 파일을 전송 하는 방법을 보여 줍니다. FTP 서버에 전송할 파일을 선택 하는 간단한.aspx 양식을 먼저 만듭니다. 다음으로 필수 메서드 및 FTP 연결을 설정 하기 위해 구성 해야 하는 FtpWebRequest 클래스의 속성 참조 하십시오. 그런 다음 로컬 파일 내용을 FtpWebRequest에 쓰려고 StreamWriter 및 StreamReader를 사용 하 여 파일을 전송할 수 있도록 하는 방법을 참조 하십시오. 마지막으로, 전송의 결과 결정 하는 FtpWebResponse 얻을 수 있습니다. 프로그래밍 방식으로 파일을 전송 하는 웹 응용 프로그램에 대 한이 방법을 사용할 수 있습니다.
