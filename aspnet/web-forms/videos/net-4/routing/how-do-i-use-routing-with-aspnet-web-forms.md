<a name="how-do-i-use-routing-with-aspnet-web-forms"></a>ASP.NET Web Forms를 사용 하 여 라우팅 작업 방법:?
====================
으로 [Chris Pels](https://twitter.com/chrispels)

이 비디오에서는 Chris Pels ASP.NET 4에서 Web Forms에 대 한 라우팅을 구현 하는 방법을 보여 줍니다. 첫째, URL 라우팅의 개념을 사이트의 실제 파일에 URL 매핑 비교 됩니다. 그런 다음 샘플 경로 URL가 global.asax에 정의 되어 파일 응용 프로그램\_시작 이벤트 처리기입니다. 경로는 URL에 입력할 수 있는 매개 변수화 된 값을 포함 합니다. 샘플 페이지 만들어집니다 및 페이지에서 경로 매개 변수 값을 추출할\_Load 이벤트 처리기입니다. 다음 두 번째 경로 여러 매개 변수 및 초기 경로와 동일한 페이지에 대 한 경로가 정의 됩니다. 페이지\_Load 이벤트 처리기 추가 경로 매개 변수 값을 추출 하 고 페이지에 전달 된 값에 따라 다른 정보를 표시 하도록 확장 됩니다.
