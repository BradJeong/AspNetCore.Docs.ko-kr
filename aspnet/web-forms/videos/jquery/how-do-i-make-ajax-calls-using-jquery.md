<a name="how-do-i-make-ajax-calls-using-jquery"></a>작업 방법: Ajax 호출 jQuery를 사용 하 여?
====================
으로 [Chris Pels](https://twitter.com/chrispels)

이 비디오 Chris Pels jQuery 라이브러리를 사용 하 여 Ajax 호출을 수행 하는 방법을 보여 줍니다. 값을 반환 하는 간단한 웹 서비스를 포함 하는 샘플 웹 사이트가 만들어집니다. 다음으로, jQuery ajax() 메서드 호출 웹 서비스에 사용 됩니다. 다양 한 ajax() 메서드 옵션 url, 데이터 형식, 데이터 등을 포함 하 여 설정 됩니다. 또한 모두 성공 및 오류 조건에 대 한 이벤트 처리기가 정의 됩니다. 다른 웹 서비스는 인수를 허용 하 고 값을 반환 한다는 그런 다음 정의 됩니다. 두 번째 ajax() 호출 웹 서비스 호출에 값을 전달 하 고 반환 데이터를 처리 하는 정의 됩니다. 마지막으로, 전역 옵션에 대 한 설치 및 ajax() 메서드 호출에 대 한 이벤트 처리기를 설명 합니다.
